# Privacy Policy

## Data Collection
Duck Translate extension does not collect, store, or share any user personal data.

## Translation Functionality
- Text selected by users is only sent to MiniMax API for translation purposes
- Translation requests are sent directly from the user's browser to MiniMax servers
- The extension does not proxy or cache any translation data

## Data Storage
- User settings (such as API Key, language preferences) are stored in the browser's local storage
- All data is stored only on the user's local device

## Third-Party Services
- This extension uses MiniMax API for translation services
- MiniMax's privacy policy applies to their services: https://www.minimax.chat/privacy

## Permission Usage
- `storage`: Only used to save user preferences
- `activeTab`: Only activates the current tab when user selects text
- `contextMenus`: Adds right-click translation menu
- Host permissions: Allows displaying translation popup on any webpage

## Contact
If you have any privacy-related questions, please contact via GitHub Issues: https://github.com/duckcodecc/duck-translate/issues

## Updates to This Policy
We may update this privacy policy from time to time. Any changes will be posted on this page.
